# wheat-fusarium-deg-analysis
Differential gene expression analysis of wheat under Fusarium stress using python.
## Project Overview

This project covers differential gene expression and RNA-seq analysis of wheat under Fusarium stress using Python and bioinformatics tools.

## Objectives
* Identify upregulated and downregulated genes
* Perform gene annotation
* Visualize differential expression using a volcano plot

## Files Included
* 'wheat_fusarium_DEG_analysis.ipynb' — Python notebook
* 'volcano_plot.png' — DEG visualization
* 'annotated_upregulated_genes.xlsx'
* 'annotated_downregulated_genes.xlsx'
* 'GSE54553_series_matrix.txt' — GEO dataset

## Tools Used
* Python
* Pandas
* Matplotlib
* GEO dataset analysis

## Results
The project identified differentially expressed genes associated with the Fusarium stress response in wheat.
